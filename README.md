# Agentic AI Lab

A working notebook of experiments, patterns, and failure modes as I
build with agentic AI systems.

## Why this exists

Agentic AI is in the awkward phase where best practices are forming in
public. I'm treating this repo as a log: what I try, what works, what
doesn't, and why.

## What you'll find here

- **Experiments** — isolated prototypes testing specific patterns
  (tool orchestration, memory strategies, evaluation harnesses)
- **Notes** — short write-ups on what I learned from each
- **Stacks I've tried** — honest opinions on frameworks, not marketing

## What you won't find

Production systems. Client work. Polished tutorials. This is my
thinking space, not a product.

## Running anything here

Each experiment folder has its own README with setup. No shared
environment — deliberately, to keep experiments isolated.
